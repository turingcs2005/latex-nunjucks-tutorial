An introduciton to LaTeX

This quick guide is based on Lindsey Fletcher's LaTeX tutorial. Learn LaTeX in 30 minutes offers another good resource.

1. Document structure
A LaTeX document has 2 parts:
  • preamble: the meta data
    • document title
    • imported packages
    • custom functions
  • document body: the content
 2. Special characters
character(s) function

\\ line break
% comment
{ } command parameters
[ ] optional parameters
& alignment
$ ... $ inline math formula

3. Basic styling

3.1 Font styles, paragraph indentation and new page

LaTeX function

3.2 Font size

Font sizes are identified by special names. The actual size is not absolute but relative to the font size declared in the \documentclass statement. (defaulted to size 10 for articles)

  1. \tiny
  2. \scriptsize
  3. \footnotesize
  4. \small
  5. \normalsize
  6. \large
  7. \Large
  8. \LARGE
  9. \huge
  10. \Huge
