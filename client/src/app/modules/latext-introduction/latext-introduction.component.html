An introduciton to LaTeX

This quick guide is based on Lindsey Fletcher's LaTeX tutorial. Learn LaTeX in 30 minutes offers another good resource.

1. Document structure
A LaTeX document has 2 parts:
  • preamble: the meta data
    • document title
    • imported packages
    • custom functions
  • document body: the content
 2. Special characters
character(s) function

\\ line break
% comment
{ } command parameters
[ ] optional parameters

$ ... $ inline math formula
